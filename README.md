# Mi Tarjeta de Perfil Web

Este es un proyecto súper sencillo de una sola página web que funciona como mi tarjeta de visita online.

## ¿Qué contiene?
* **Una imagen:** Mi "marca de agua" personal y tonta para darle personalidad.
* **Una descripción:** Un párrafo breve sobre quién soy y qué estudio.
* **Diseño Flexible:** Implementación de CSS Flexbox para centrar la tarjeta y organizar el contenido interno de forma responsiva.
